HTML에 스타일을 적용하기

HTML을 스타일을 변경하는 언어를 CSS(종속형 시트/Cascading Style Sheets)라고 합니다. 물론 몰라도 CSS를 사용하는데 아무런 문제 없습니다. 중요한건 CSS를 이용하면 HTML의 태그를 더 다채롭게 업그레이드 할 수 있다는 것입니다.

흰 배경에, 검정 글씨로만 이루어진 웹 페이지는 사실 많지 않아요. 각자 자신만의 스타일을 가지고 있습니다. 예를들어 <h1>의 색상을 파랑색으로 변경하고 싶다거나, 본문의 글씨 크기를 조금 더 크게 키우고 싶을 때 어떻게 해야 할까요? 이번 시간에는 HTML에 스타일을 적용하는 방법을 알아 볼 겁니다.

첫번째, HTML 태그에 직접 스타일을 부여한다.

h1태그의 배경을 검정으로 바꾸고, 색상을 하얀색으로 하고싶을 때, 할 수 있는 첫번 째 방법은 해당 태그에 직접 스타일을 부여하는 것입니다. 위와 같이 <h1과 > 사이에 style=""을 추가합니다. 따옴표 사이에 스타일을 부여하게 됩니다. 

<h1 style="background-color: black;">

좀 쉽게 이해하기 위하여 한글로 써볼게요. 

<h1 스타일="배경-색: 검정;">

스타일을 부여하는 방법은 모두 같습니다. 왼쪽에 무엇을 :항목을 작성하고, 한 칸띄고, 어떻게;라고 작성하는 방식이죠. 중요한것은 반드시 ;마크로 이 스타일을 마무리 지어야한다는 것 입니다. tag에서 /와 같은 의미 입니다. 아래의 예시를 통해서 확인해봅시다. h1태그의 컬러를 yellow로 변경해 보세요.

두번째, head 태그 안에 style 태그를 추가하기

만약 12개의 <h1>가 있을 때, 이 <h1>의 컬러를 blue로 변경해야 한다면 어떻게 할까요? 모든 태그에 style="color: blue;"라고 하면 될 겁니다.

하지만 문제는 12번 써야 한다는 것입니다. 만약 12개가 아니고, 100개라면 100번을 써야할 거예요. 그런데, blue로 해놓고, 보니 사실은 red가 더 잘 어울리는 것 같다면 다시 100면을 수정해야하는 상황에 빠질 겁니다.

그런 것을 이미 겪은 아주 오래전 선배 개발자들이 <style>라는 것을 만들었습니다. 그리고 이 <style>를 <head>안에다가 숨겨 놓고 사용하기 시작했어요. <head>에는 이처럼 보이지 않는 어떤 태그들을 넣어둘 수 있습니다. <style>태그도 그중 하나죠. 

<head>
  <style>
    h1 { color: blue; }
  </style>
</head>
<body>
  ...
</body>

마지막, CSS 스타일들만 담겨있는 별도의 공간을 만들기

위에 style태그를 활용해서 여러개의 태그를 컨트롤 했습니다. 만약 h1태그가 10개씩 들어있는 HTML문서가 50개인 웹 서비스가 있다면 어떻게 해야 할까요? 실제로 우리가 이용하는 웹 서비스들은 더 많은 수의 html파일로 이루어져 있거든요.

따라서 50개의 HTML에 동시에 적용할 수 있는 CSS 연결고리가 필요합니다. 하지만 우리가 50개의 HTML 문서를 만들면서 할 수 있는건, 지금은 없기 때문에 여기서는 배우지 않겠습니다. 대신! Codepen에 있는 CSS 항목을 작성하는 것으로 CSS를 더 익숙해 지도록 합시다. 아래 예시의 상단에 에서 CSS 버튼을 클릭해 보세요. 마찬가지로 h1 태그의 컬러를 orange로 변경해 봅시다.
