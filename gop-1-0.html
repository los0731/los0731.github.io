시작하기 전에

우리 코스는 HTML과 CSS로 무언가를 만들겁니다. 하지만 실제로 사용할 수 있는 것들을 만들겁니다. 예를들어, 모바일 명함이나, 생일 축하카드 같이요. 카카오톡이나 이메일로 보낼 수 있도록 우리가 만든 웹 페이지를 주소(url)형태로 보낼 거예요.

하지만 그렇게 하기 위해서는 좀 어려운 과정이 필요합니다. 텍스트 에디터를 설치해야 하고, 웹 서버를 구축해야합니다. 그건 우리가 배우려는 것 이상의 난의도죠. 따라서 이 모든것을 대신 해줄 수 있는 도구를 이용할 겁니다. Codepen 입니다. 이 코스를 진행하기 전에 Codepen에 계정을 만들어 봅시다.

물론 무료입니다!

코드팬 홈페이지로 이동

  • 1. 코드팬 홈페이지로 이동합니다. 우측 상단에 Sign Up 버튼을 클릭합니다.

  • 2. 우리는 무료로 이용할 것이니까, 오른쪽에 Join CodePen Free 버튼을 클릭합니다.

  • 3. 이름, 닉네임, 이메일, 비밀번호, 로봇이 아닙니다 등을 입력한 후 Submit 버튼을 클릭합니다.

  • 4. Save & Continue 버튼을 클릭합니다.

  • 5. 튜토리얼이 진행되네요. 하지만 우리는 아직 아무것도 알지 못하는 단계이므로 I'd rather not take to tour right now 링크를 클릭합니다.

  • 6. 이제 계정이 정상적으로 만들어 졌습니다. 상단의 검은 배경의 3칸이 각각 html/css/javsscript를 입력하는 에디터 입니다. 저곳에 코드를 입력하면 아래 흰 배경에 결과물이 나오게 됩니다. 홈 버튼을 클릭해 봅시다!

  • 7. 이곳이 www.codepen.io로 접속하면 나오는 화면입니다. 앞으로 새로운 html 문서를 시작할 때 우측 상단의 Create버튼을 클릭해서, New Pen을 클릭하시면 됩니다!
