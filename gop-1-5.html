가장 기초적인 태그들

아래의 태그들은 가장 많이 쓰면서도 가장 기초적인 태그들 입니다. 다음 챕터 부터 하나하나 살펴 볼 것이기 때문에 외울 필요 없어요. 가벼운 마음으로 이런 태그가 있다는 것만 정도만 알아둡시다.

  • 태그가 없다면?

    태그가 없는 상태로 글을 쓰면 어떻게 될까요? 글은 잘 나오는데, 줄바뀜도 안되고 영 보기가 쉽지 않네요.

    See the Pen html tag - none by los0731 (@los0731) on CodePen.

  • <h1>

    Heading(헤딩/제목) 태그는 말 그대로 제목 태그입니다. 문서의 제목 글에 이 태그를 쓰죠. h1~h6까지 있으며, 숫자가 작아질수록 크기가 커집니다. <h1>태그가 끝날때에는 자동으로 줄바뀜이 적용됩니다.

    See the Pen html tag - h1~h6 by los0731 (@los0731) on CodePen.

  • <p>

    Paragraph(패러그래프/문단) 문단 태그는 내용이 같은 글줄 단위로 사용합니다. 뉴스 기사나 블로그의 본문이 바로 이 <p>태그 입니다. <p>태그가 끝날때에는 자동으로 줄바뀜이 적용됩니다.

    See the Pen html tag - p by los0731 (@los0731) on CodePen.

  • <br>

    줄을 바꾸고 싶나요? 줄을 바꾸고 싶은 부분에 <br>을 입력하세요. br은 single line break의 줄인 말로, 줄바꿈 이란 뜻입니다.

    See the Pen html tag - br by los0731 (@los0731) on CodePen.

  • <b>

    <b>는 bold, 두꺼운이라는 의미입니다. <i>는 italic, 기울어진이라는 뜻입니다. 이렇게 글씨를 꾸며주는 태그들도 있어요.

    See the Pen html tag - bold by los0731 (@los0731) on CodePen.

  • <img>

    이미지는 가장 많이 사용하는 요소(Element)중 하나죠! 이미지는 어떻게 쓸까요? 

    <img src="이미지 주소 (url)" alt="이미지의 이름" width="이미지 넓이(숫자)">

    먼저, 이미지 태그는 예외적으로 닫는 태그가 없는데, <p>처럼 무언가를 감싸고 그 안에 글씨나 다른 요소를 넣을 필요가 없기 때문입니다.

    이미지 태그에 사용된 width="300"의 300 뒤에 아무것도 입력하지 않으면 기본단위인 px로 적용됩니다. 하지만 300%라고 할 경우, %로 적용할 수 있어요.

    See the Pen html tag - img by los0731 (@los0731) on CodePen.

다음 시간에는 <h1>와 <p>를 활용해서 짧은 편지를 써봅시다.
