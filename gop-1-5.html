HTML은 안테나/머리/몸.

짜잔! HTML은 이렇게 생겼습니다. (어디서 많이 봤나요?) :-) HTML은 안테나-머리-몸으로 구분되어 있습니다. 각각 맡은 역할이 있죠. 어떤 역할들을 하는지 이제부터 알아봅시다!

  • 안테나 부분 <!DOCTYPE html>
    모든 html 문서의 최 상단에는 "내가 HTML 이다!"라는 의미를 담은 코드를 한 줄 넣게 됩니다. 정말 안테나 같죠? 예컨대 (외울필요없는!) <!DOCTYPE html> 이런 코드들이죠. 저 코드가 첫 줄에 작성되어 있으면 브라우저는 이 파일이 "아. 너 HTML이구나?" 하고 인식합니다.
  • 머리 부분 <head></head>
    머리 부분에는 이 HTML 문서가 어떤 이름을 가지고 있고, 어떤 특이한 속성들을 가지고 있는지를 담아둡니다. 이곳에 곧 만나게 될 CSS 파일들을 불러올 수 있는 코드를 넣기도 하고, 모바일에서도 1:1 비율로 잘 보이도록 하는 코드도 추가할 수 있죠.
  • 몸 부분 <body></body>
    우리가 웹페이지를 보면서 눈에 보이는 모든 것을 작성하는 공간이 바로 이곳입니다. 이곳에 로고 이미지도 넣고, 버튼도 넣고, 글씨도 넣죠. 우리는 HTML의 몸만 봐왔던 것이에요.

우리는 안테나 부분과, 머리 부분은 거의 사용하지 않을 것입니다. 우리는 오로지 몸 부분에 98%의 코드를 작성하게 될 것이에요.

자, 이제 다음 시간부터는 본격적으로 HTML 코딩을 해봅시다.
