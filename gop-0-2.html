마지막으로

이제 공식 요청서를 만들었으니, 실제로 활용을 해야겠죠? 이 웹페이지를 친구에게 보내기 위해서는 작업 중인 Codepen을 Full Page Mode로 변경 후 URL을 복붙 하기만 하면 됩니다. 아래의 이미지를 보고 하나하나 따라 해 봅시다.

  • 1. 우리가 코드를 작성하던 Codepen 에디터 가운데 위쪽에 SAVE버튼이 있을 겁니다. 그것을 클릭하세요.

  • 2. 그럼 영어로 뭐라뭐라 나오는데, free! 라는 단어가 보이실 겁니다. 맞아요. 무료입니다. 여기서 계정을 생성하세요. 이메일 인증이 필요하니 이메일 인증도 해줍시다.

  • 3. 계정을 생성하셨나요? 오른쪽 위에 Change View버튼이 보일겁니다. 그걸 클릭하면 나오는 목록에서 Full Page를 클릭합니다.

  • 4. 이 모습이 Full page mode 입니다. 다시 에디터를 활성화 하고싶다면 Change View버튼을 클릭하여 Editor View항목을 선택하면됩니다. 자. 이제 full page mode로 변경하고, 브라우저 상단에 url을 복사 후 친구에게 카톡 메시지로 보냅시다.
