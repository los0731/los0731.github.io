애니메이션 카드

이번 시간에는 애니메이션이 첨가된 온라인 카드를 만들어 봅시다. 저는 이제 곧 30세가 되는 친구 지영이를 위해 온라인 축하 카드를 만들어 봤어요. 지영이에게는 카톡으로 url을 보낼 예정이구요, 그 url을 클릭했을 때, 생일 축하 카드가 보이도록 할 겁니다.

최종 결과물

See the Pen A card that opens when you click it. by los0731 (@los0731) on CodePen.

사용된 요소들

HTML Tags
└ <meta charset="utf-8"> /* utf-8 인코딩, 한글을 보이도록 해준다. */
└ <meta name="viewport" content="width=device-width, user-scalable=no"> /* 모바일에 최적화된 해당도로 맞춰준다. 자세히보기*/
└ <h1></h1>
└ <p></p>
└ <a href="https://link.com"></a>
└ <div></div>
└ <img src="https://sourse-url.com">

CSS Properties
└ display: flex; /* flex box 자세히보기 */
└ justify-content: center;
└ align-items: center:
└ margin: 0;
└ padding: 20px;
└ background: #ffffff url('https://sourse-url.com') no-repeat center;  /* src 앞의 컬러 값은 이미지 밑의 컬러, 이미지 로딩시 보이게 됩니다. */
└ background-size: cover; /* 배경 이미지를 영역에 딱 맞게 채워줍니다. */
└ background-color: white;
└ width: 100px;
└ max-width: 200px; /* width가 최대 200px을 넘지 않습니다. 그리고 max-height */
└ position: fixed; /* 위치를 고정, 스크롤해도 그 자리에서 움직이지 않습니다. 자세히보기 */
└ top: 50% /* top에서 50%위치 만큼 아래에 위치합니다. */
└ transition: 1s; /* 어떤 변화든 1초동안 부드럽게 진행됩니다. 자세히보기 *
└ transform: translateX(-50%) translateY(-50%); /* 트렌스폼: 이동, 회전, 시전, 크기 등을 변형합니다. 자세히보기 *
└ border-radius: 10px; /* 모서리를 둥글게! */
└ box-shadow: 0 40px 80px -40px rgba(50,50,80,.8); /* 그림자, 값은 순서대로 x축, y축, 흐림, 크기입니다. */
└ overflow-y: scroll; /* 내용이 세로영역보다 길어지면, 스크롤 되도록 합니다. */
└ -webkit-overflow-scrolling: touch; /* 모바일에서 스크롤이 부드럽게! */
└ .class::-webkit-scrollbar { display: none; } /* webkit 계열(크롬,파폭 외)에서 스크롤바를 보이지 않도록 합니다. */
└ font-size: 18px;
└ line-height: 26px;
└ font-weight: 700;
└ display: none;

  • 튜토리얼 시작

  • 우선, 여기서부터 시작합니다. 아직 아무 코드도 작성하지 않았어요. 이 카드는 모바일에서 보여질 수도 있기 때문에, 모바일에서도 해상도가 최적회 되도록 나오는 코드를 head태그 안에 추가합니다.

    <head>
  <meta name="viewport" content="width=device-width, user-scalable=no">
</head>
