애니메이션 카드

애니메이션이 첨가된 온라인 카드를 만들어 봅시다. 저는 이제 곧 30세가 되는 친구 지영이를 위해 온라인 축하 카드를 만들겁니다. 지영이에게는 카톡으로 url을 보낼 예정이구요, 그 url을 클릭했을 때, 생일 축하 카드가 보이도록 할 겁니다.

코드를 작성하기 앞서, 스케치를 해보는 것은 좋은 습관입니다.

저는 앞면, 뒷면의 두장의 카드를 따로 만든 후, 이걸 겹쳐서 앞면이 위로 올라가도록 만들 생각입니다. 그리고, 마우스를 올리거나, 터치했을 때 앞면이 옆으로 스륵 밀려나는 효과를 주면 되겠네요.

그럼 코드를 작성해 봅시다.

  • 언제나 가장 먼저 하는것은 head해드에 적절한 코드를 넣어주는 것입니다. 저는 title태그를 먼저 넣습니다.

    중/고급자와 초급자의 차이는 역시 내가 무엇을 모르는지 아는것과 모르는 것의 차이 인것 같아요. 저는 모바일에서는 모바일에 최적화 되어야하는 코드가 필요하다는 것을 알고 있습니다. 하지만 그 코드를 외우지는 못해요. 그래서 언제나 html 모바일 최적화라고 구글에 검색합니다.

    검색해 보셨나요? 상단의 몇개 개시글들을 살펴보면 head태그 안에 어떤 meta태그를 추가하라고 하는걸 찾으셨나요? 그대로 복붙 하겠습니다.

    See the Pen animation card - step 1 by los0731 (@los0731) on CodePen.

  • 카드가 올라갔을 때 더 아름답게 보일 수 있도록, 배경에 멋진 이미지를 채우고 싶어요. 다만, 이미지를 고르는것이 부담스러울 거예요. 우리는 디자이너가 아니니까요. 그럼 아래를 따라서 해봅시다. 이건 현디자이너 친구가 알려준 방법입니다.

    1. unsplash.com에 접속합니다.
    2. 여기 있는 모든 이미지는 전부 저작권이 무료입니다. 게다가 전부 퀄리티가 좋아요. 마음에 드는 사진을 아무거나 고릅니다.
    3. 이미지를 클릭해서 나오는 큰 이미지를 우 클릭해서 나오는 메뉴에서이미지 주소복사를 합니다.
    4. 이미지 url을 HTML/CSS에서 사용합니다. 끝.

  • HTML의 body태그에는 기본적으로 margin: 8px;이 적용되어 있다고 합니다. 그래서 배경을 전부 사용해야할 경우에는 이 마진을 제거해 줘야합니다. 그리고 이 background 속성을 활용하여 위에서 선택한 이미지를 채웁니다. 그리고 background-size를 cover로 지정해 줍니다.

    여기까지 했을 경우, 이미지에 따라 다르겠지만, 이미지가 배경에 다 차지 않게 될거예요. 따라서, body태그의 높이를 브라우저의 높이로 고정해줘야합니다. 그것은 height: 100vh;라고 합니다. vh는 viewport height의 약자로, viewport 쉽게 말해 브라우저에 웹 페이지가 표시되는 영역이라고 생각하시면 편리합니다. 좀더 자세히 알고 싶다면 여기을 클릭하세요.

사용된 요소들

HTML Tags
└ <meta charset="utf-8"> /* utf-8 인코딩, 한글을 보이도록 해준다. */
└ <meta name="viewport" content="width=device-width, user-scalable=no"> /* 모바일에 최적화된 해당도로 맞춰준다. 자세히보기*/
└ <h1></h1>
└ <p></p>
└ <a href="https://link.com"></a>
└ <div></div>
└ <img src="https://sourse-url.com">

CSS Properties
└ display: flex; /* flex box 자세히보기 */
└ justify-content: center;
└ align-items: center:
└ margin: 0;
└ padding: 20px;
└ background: #ffffff url('https://sourse-url.com') no-repeat center;  /* src 앞의 컬러 값은 이미지 밑의 컬러, 이미지 로딩시 보이게 됩니다. */
└ background-size: cover; /* 배경 이미지를 영역에 딱 맞게 채워줍니다. */
└ background-color: white;
└ width: 100px;
└ max-width: 200px; /* width가 최대 200px을 넘지 않습니다. 그리고 max-height */
└ position: fixed; /* 위치를 고정, 스크롤해도 그 자리에서 움직이지 않습니다. 자세히보기 */
└ top: 50% /* top에서 50%위치 만큼 아래에 위치합니다. */
└ transition: 1s; /* 어떤 변화든 1초동안 부드럽게 진행됩니다. 자세히보기 *
└ transform: translateX(-50%) translateY(-50%); /* 트렌스폼: 이동, 회전, 시전, 크기 등을 변형합니다. 자세히보기 *
└ border-radius: 10px; /* 모서리를 둥글게! */
└ box-shadow: 0 40px 80px -40px rgba(50,50,80,.8); /* 그림자, 값은 순서대로 x축, y축, 흐림, 크기입니다. */
└ overflow-y: scroll; /* 내용이 세로영역보다 길어지면, 스크롤 되도록 합니다. */
└ -webkit-overflow-scrolling: touch; /* 모바일에서 스크롤이 부드럽게! */
└ .class::-webkit-scrollbar { display: none; } /* webkit 계열(크롬,파폭 외)에서 스크롤바를 보이지 않도록 합니다. */
└ font-size: 18px;
└ line-height: 26px;
└ font-weight: 700;
└ display: none;
